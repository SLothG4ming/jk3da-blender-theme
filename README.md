# JK3DA Blender Theme

A dark, warm Blender theme with a signature **orange (#E75A0A)** accent color —
built and maintained by [Jonas Kircher (JK3DA)](https://jk3da.com).

## Features

- Warm dark backgrounds (~#171614 to #232220)
- Orange accent throughout: selections, active objects, buttons, gizmos, keyframes, markers
- Orange active state for toggle & radio buttons (Snap, Pivot, resolution presets…)
- Orange menu item hover (Shift+A Add menu, right-click context menus)
- Orange Loop Cut preview line
- Orange active viewport border
- Readable white text on orange Outliner selection backgrounds
- Rounded buttons

## Compatibility

| Blender | Status |
|---------|--------|

## Installation

### Via Blender Extensions (recommended)
1. In Blender: **Edit → Preferences → Get Extensions**
2. Search for **JK3DA Theme** and click **Install**

### Manual install
1. Download `JK3DA.xml` from this repository
2. In Blender: **Edit → Preferences → Themes**
3. Click the folder icon → **Install…**
4. Select `JK3DA.xml`
5. Done!
